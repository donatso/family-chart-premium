## 🚀 Quick Start

The fastest way to get started is by using our [**Visual Creator**](https://donatso.github.io/family-chart-premium/playground/creator/). It allows you to design your lineage visually and export the implementation code directly for React, Vue, Svelte, or Vanilla JS.

Alternatively, you can browse any of our [**Live Demos**](https://donatso.github.io/family-chart-premium/playground/home/) and copy the source code to use as a starting point for your project.
